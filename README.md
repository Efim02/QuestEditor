# QuestEditor
The quest editor for a game project.

Расширения поддерживающихся файлов: xlsx

Идея: формировать квестовые задания с помощью desktop приложения

Программа: программа редакториует конкретный квест.

Задачи:
	- Написать модели (структуры данных), для представления квестов
	- Добавить сериализацию и дессириализацию моделей в файл.xslx
	- Написать view и view-model для model-квестов
	- Написать представление иерархичной структуры квестов (схематично)
	- Написать взаимодействие для работы с иерархичной структурой данных (схематично)
	- Добавить кнопки открыть, создать, сохранить, выйти
